# logo-kementerian

Kumpulan logo kementerian

## Tentang dan Tujuan

Ini adalah kumpulan logo kementerian di Indonesia. Tujuan repository ini adalah untuk mempermudah bagi siapa saja yang membutuhkan terutama untuk persuratan. 

Tipe file logo adalah PNG tidak transparan dan resolusi tidak tinggi. Penamaan file logo menggunakan kode yang ada di [Lampiran Peraturan Menteri Keuagan Nomor 101/PMK.02/2011](https://jdih.kemenkeu.go.id/fulltext/2011/101~PMK.02~2011PerLamp%2520I.htm).

Semoga bermanfaat.
